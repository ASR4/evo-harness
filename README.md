# EvoHarness

Automatically improve LLM agent harnesses through evolutionary search.

Inspired by [Meta-Harness](https://arxiv.org/abs/2603.28052) (Lee et al., 2026).

## Installation

```bash
pip install -e ".[dev]"
```

## Quick Start

```bash
evo init --template basic my-project
cd my-project
evo run
```
